### Step 2: Supported Formats
- `.jpg` / `.jpeg` (most common)
- `.png` (for images with transparency)
- `.webp` (best performance, smaller file size)

### Step 3: Image Optimization Tips
- **Compress images** before adding them (use tools like TinyPNG, ImageOptim, or Squoosh)
- **Recommended file sizes:**
  - Hero image: Under 500KB
  - Services image: Under 500KB
- **Aspect ratio:** 16:9 works best for backgrounds

### Step 4: After Adding Images
1. Save your images in this folder with the correct names
2. Refresh your browser (or restart `npm run dev` if needed)
3. The images should automatically appear!

## Current Image Usage

## Troubleshooting

If images don't appear:
1. Check that file names match exactly (case-sensitive)
2. Make sure images are in `public/images/` folder (not `src/assets/images/`)
3. Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)
4. Check browser console for any errors
